# Restaurant-Data-Analysis
# 🍽️ Restaurant Data Analysis Dashboard

## 📌 Project Overview
This project analyzes restaurant data to uncover insights related to customer preferences, location trends, and business features such as online delivery and table booking. The objective is to support data-driven decisions in the food and hospitality industry.

This project was completed during a Data Analyst Internship, applying SQL and Power BI to solve real-world business problems in the restaurant domain.

## 🎯 Objectives
* Analyze restaurant performance based on ratings and customer votes
* Identify high-demand locations and cuisines
* Evaluate the impact of online delivery and table booking on customer engagement

## 📊 Dataset Details
* 📁 Records: 9,000+ restaurants
* 📌 Key Features:
  * Restaurant Name & Location
  * City & Locality
  * Cuisines
  * Ratings & Votes
  * Online Delivery & Table Booking
  * Latitude & Longitude

## 🛠 Tools & Technologies
* SQL (Data extraction and querying)
* Power BI (Dashboard and visualization)
* MS Excel (Data preprocessing)
  
## 🔧 Data Processing
* Cleaned dataset by handling missing values and inconsistent entries
* Standardized location and cuisine data for accurate analysis
* Prepared data for geospatial and trend analysis
  
## 📈 Key Analysis Performed
* Conducted **city-wise and locality-wise analysis** to identify high-demand areas
* Performed **geospatial analysis (latitude & longitude)** to understand restaurant distribution
* Analyzed **cuisine trends** to identify popular food preferences
* Evaluated the impact of **online delivery and table booking** on ratings and customer votes
  
## 📊 Dashboard Features
* Interactive Power BI dashboard
* Filters for city, cuisine, and restaurant features
* Visualizations for ratings, votes, and location trends
* Comparative analysis across different regions
  
## 🔍 Key Insights
* Identified **top-performing cities and localities** based on ratings and customer engagement
* Restaurants offering **online delivery showed higher customer reach and votes**
* Certain cuisines dominate in specific regions, indicating **localized customer preferences**
* High-rated restaurants are concentrated in **key urban areas**

## 🚀 Project Outcome
This project demonstrates the ability to:
* Analyze large-scale real-world datasets
* Perform geospatial and trend analysis
* Build interactive dashboards for business insights
